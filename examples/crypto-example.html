FCS API - Crypto Example

Configuration: Open src/FcsConfig.js and set your authentication:
  • authMethod = 'access_key' - Set your API key in accessKey
  • authMethod = 'ip_whitelist' - No key needed if your IP is whitelisted
  • authMethod = 'token' - Set publicKey, token, and tokenExpiry
